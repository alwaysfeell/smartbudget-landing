SmartBudget

Інформаційна система підтримки прийняття рішень щодо споживчих витрат

Система аналізу витрат, прогнозування бюджету та персоналізованих фінансових рекомендацій на основі правила 50/30/20 та сценарного моделювання.

Python Flask SQLite DSS MVC Bootstrap 5

Про роботу

Актуальність теми

В умовах зростання фінансової нестабільності та складності управління особистими фінансами більшість громадян не мають зручних інструментів для аналізу своїх витрат та прийняття обґрунтованих фінансових рішень. Наявні рішення або надто складні, або не адаптовані до українських реалій.

SmartBudget вирішує цю проблему через побудову доступної веб-системи з елементами DSS, що аналізує витрати, виявляє аномалії, будує прогнози та надає персоналізовані рекомендації.

Мета та завдання

01

Мета дослідження

Розробити інформаційну веб-систему підтримки прийняття рішень, що автоматизує аналіз споживчих витрат та формує обґрунтовані фінансові рекомендації для кінцевого користувача.

Основні завдання

  1. Провести аналіз предметної галузі управління споживчими витратами та наявних DSS-рішень
  2. Спроектувати архітектуру MVC-застосунку з модулями аналітики, прогнозування та рекомендацій
  3. Реалізувати модуль DSS з правилом 50/30/20, сценарним аналізом та оцінкою якості даних
  4. Розробити інтерактивний дашборд із графіками, статистикою та аналізом цін
  5. Провести тестування системи та оцінити якість рекомендацій

Методологія

⬡

Архітектура MVC

Розділення на шари Model (бізнес-логіка), View (Jinja2 + Bootstrap), Controller (Flask Blueprints). Чітка відповідальність кожного компонента.

◈

Модуль DSS

Правило 50/30/20 (потреби/бажання/заощадження), прогнозування трендів, сценарний аналіз та оцінка якості вхідних даних.

◎

Аналіз даних

Статистичні звіти, графіки витрат, аналіз цін по категоріях, виявлення аномалій та персоналізовані рекомендації.

▦

Зберігання даних

SQLite база даних з ініціалізацією схеми та seed-даними. Структурований доступ через database.py з чітким API.

Очікувані результати

Функціональна система

Повноцінний веб-застосунок із дашбордом, модулями витрат, аналізу, рекомендацій, DSS, профілю та фінансових цілей.

Практична цінність

Система надає реальну допомогу користувачам у контролі бюджету, виявленні надмірних витрат та плануванні фінансів.
